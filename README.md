# Horizontal Dodg

# Control
- 좌클릭 = 플레이어 이동

# Information
1. 장애물을 피하면 1점이 증가합니다.
2. 플레이어가 벽에 닿으면 2점이 증가합니다.

# How to play video games
- 장애물을 피해 최대한 높은 점수를 얻으세요.
